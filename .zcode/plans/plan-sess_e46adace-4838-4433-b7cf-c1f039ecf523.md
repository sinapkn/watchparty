## بهبود UX دکمه ویس چت

### مشکل فعلی:
- آیکون جدا + دکمه متن جدا + سایز کوچیک
- کلیک سخت، تجربه بد

### راه‌حل: یه دکمه یکپارچه بزرگتر (مثل زوم، دیسکورد)

**تغییر در `src/app/room/[id]/page.tsx` — خط ۱۴۵-۲۱۲:**
حذف آیکون جدا + دکمه متن. تبدیل به یه دکمه بزرگتر با آیکون + متن کنار هم:

```
[ 🎤 وصل ]   ← سبز = میکروفون فعال
[ 🔇 خاموش ] ← قرمز = میکروفون قطع
[ ویس... ]   ← خاکستری = در حال اتصال
[ 🟢 نقطه ]  ← هنگام حرف زدن، نقطه سبز بالای دکمه
```

- کلیک ساده = toggle mute/unmute
- سایز بزرگتر (py-1.5 + px-3)
- متن هم هست: "وصل" / "خاموش" / "ویس..."
- غیرفعال بودن در زمان اتصال (cursor-wait)
